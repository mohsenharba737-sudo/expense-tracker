🇬🇧 README (English Version)
Expense Tracker – Modern English & Arabic UI
A lightweight and elegant web application for tracking income and expenses.
Designed with a clean English & Arabic interface, smooth interactions, and automatic data saving using LocalStorage.
The app is fully client‑side and requires no backend or external libraries.

✨ Features
Add income

Add expense

Automatic balance calculation

View total income and total expenses

Delete individual transactions

Clear all transactions

Dark / Light mode toggle

Data saved automatically in LocalStorage

Modern Glassmorphism UI

Fully responsive on mobile and desktop

📁 Project Structure
Code
project/
│
├── index.html      # Main UI
├── style.css       # Styling and theme
└── app.js          # App logic (JavaScript)
🚀 How to Run
Download the three files:
index.html, style.css, app.js

Place them in the same folder

Open index.html in any modern browser

No server, no installation, no dependencies — it works instantly.


**direct link**


https://mohsenharba737-sudo.github.io/expense-tracker/



🧠 How It Works
Choose the transaction type (income or expense)

Enter a description

Enter the amount

Click Add Transaction

The app updates:

Current balance

Total income

Total expenses

Transaction list

You can also:

Delete a specific transaction

Clear all transactions

Switch between dark and light mode

💾 Data Storage
All data is stored locally using:

Code
localStorage
This means:

Your data stays even after closing the browser

No internet connection is required

No data is uploaded anywhere
